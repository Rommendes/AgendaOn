# 💇 AgendaOn

Sistema para gerenciamento de agenda, clientes e financeiro voltado para profissionais da beleza.

O AgendaOn foi desenvolvido para simplificar a rotina do salão, oferecendo uma interface intuitiva, organizada e eficiente para o gerenciamento de atendimentos, clientes e recebimentos.

---

## ✨ Funcionalidades

- 📅 Agenda de atendimentos
- 👥 Cadastro de clientes
- 💰 Controle financeiro
- 💳 Registro de pagamentos
- 📲 Lembretes de agendamentos
- 📊 Histórico de atendimentos
- 🔒 Autenticação de usuários

---

## 🛠️ Tecnologias

## 🚀 Como executar o projeto

### Pré-requisitos

- Git
- Node.js (recomendado via NVM)

### Clonar o repositório

```bash
git clone git@github.com:Rommendes/AgendaOn.git
cd AgendaOn
```

### Instalar as dependências

```bash
npm install
```

### Executar o projeto

```bash
npm run dev
```
